# Data Attribution and Citations

This document provides complete attribution for all thermophysical property data used in the ChemEng Toolbox compound database.

## Primary Data Sources

### CoolProp Thermophysical Property Library

**License**: MIT License

**Citation**:
> Bell, I.H., Wronski, J., Quoilin, S., and Lemort, V. (2014).
> Pure and Pseudo-pure Fluid Thermophysical Property Evaluation and the
> Open-Source Thermophysical Property Library CoolProp.
> *Industrial & Engineering Chemistry Research*, 53(6), 2498-2508.
> DOI: 10.1021/ie4033999

**Website**: http://www.coolprop.org

**GitHub**: https://github.com/CoolProp/CoolProp

**Description**: CoolProp is an open-source thermophysical property library that uses state-of-the-art reference equations of state for pure and pseudo-pure fluids. All properties are derived from peer-reviewed literature and validated against NIST REFPROP.

### NIST Chemistry WebBook

**License**: Public Domain (U.S. Government Work)

**Citation**:
> Linstrom, P.J. and Mallard, W.G., Eds.
> *NIST Chemistry WebBook, NIST Standard Reference Database Number 69*.
> National Institute of Standards and Technology, Gaithersburg MD, 20899.
> https://webbook.nist.gov (retrieved December 29, 2025)

**Website**: https://webbook.nist.gov/chemistry/

**Description**: The NIST Chemistry WebBook provides access to data compiled and distributed by NIST under the Standard Reference Data Program. We use NIST data for validation of CoolProp-extracted properties.

## Validation Methodology

All compound data in this database has been validated against NIST reference values using automated test suites. Validation tolerances:

- **Critical Temperature**: ±0.01% (relative)
- **Critical Pressure**: ±0.01% (relative)
- **Critical Density**: ±0.1% (relative)
- **Acentric Factor**: ±1% (relative)
- **Normal Boiling Point**: ±0.1% (relative)

See `packages/core/tests/validation/` for validation test source code.

## Contributing

To add a new compound with proper attribution:

1. Use the provided extraction utilities from CoolProp
2. Include NIST WebBook URL for validation
3. Document the equation of state and primary reference
4. Add validation tests against NIST values
5. Update this ATTRIBUTION.md file

See `packages/core/src/chemeng_core/data/compounds/README.md` for details.
